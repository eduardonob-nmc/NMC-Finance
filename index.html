Guia completo de uso — NMC Personal Finance v20

🏠 O que é o NMC Personal Finance System?

O NMC Personal Finance System é um aplicativo completo de controle financeiro pessoal, desenvolvido pela Nobrega Mall Consultant. Funciona diretamente no navegador do seu celular ou tablet — sem precisar de internet para operar. Todos os dados ficam salvos no próprio dispositivo com criptografia AES‑256.

Com ele você registra receitas e despesas, controla cartões de crédito, acompanha financiamentos, gerencia investimentos, define metas financeiras e concilia extratos bancários — tudo em um só lugar.

🔐 Primeiro Acesso — Login

Na primeira vez que abrir o sistema, será solicitado um nome de usuário e uma senha. Guarde bem essas informações — elas criptografam todos os seus dados. Se esquecer a senha, os dados não poderão ser recuperados (exceto via backup exportado).

  • O usuário padrão inicial é admin com senha definida por você
  • Você pode criar outros usuários em Usuários no menu
  • Cada usuário tem seus próprios dados criptografados

⚙️ Configuração Inicial — Por onde começar?

Antes de lançar qualquer movimentação, configure as bases do sistema na aba Cadastros:

  • 1. Bancos / Contas: Cadastre suas contas (ex: "Nubank CC", "Itaú Poupança"). Informe o saldo atual da conta no campo Saldo Inicial — esse valor é a base do cálculo de saldo. Se sua conta tiver cheque especial, informe o limite.
  • 2. Centros de Custo: Crie categorias para classificar seus lançamentos (ex: Alimentação, Moradia, Saúde, Salário). Defina o tipo: Receita, Despesa ou Ambos.
  • 3. Formas de Pagamento: Cadastre as formas que usa (PIX, Débito, Boleto, Dinheiro etc.).

📝 Como Lançar Receitas e Despesas

Acesse a aba Lançamentos e clique em + Novo Lançamento. Preencha:

  • Tipo: Receita (dinheiro que entra) ou Despesa (dinheiro que sai)
  • Data: Data da movimentação
  • Banco / Conta: De qual conta saiu ou entrou o dinheiro
  • Centro de Custo: Categoria do lançamento
  • Forma de Pagamento: Como foi pago
  • Valor: Digite o valor (ex: 1.500,00)
  • Descrição: Texto livre para identificar o lançamento
  • Pago / Pendente: Marque se já foi efetivado ou é um agendamento futuro

💡 Compra parcelada no cartão: Selecione o cartão e informe o número de parcelas — o sistema gera todas as parcelas automaticamente nas datas corretas.

💡 Transferência entre contas: Selecione o tipo "Transferência" e informe a conta destino — o sistema cria um lançamento de saída e outro de entrada automaticamente.

🔍 Filtros e Pesquisa de Lançamentos

Na aba Lançamentos, use a barra de filtros para localizar rapidamente qualquer movimentação:

  • Busca por texto: Digite descrição, valor ou categoria
  • Tipo: Filtre por Receita, Despesa, Financiamento ou Transferência
  • Banco: Visualize apenas uma conta
  • Centro de Custo: Filtre por categoria
  • Período: Defina data início e fim
  • Status: Veja apenas pagos ou pendentes

💳 Cartões de Crédito

Cadastre seus cartões em Cartões informando nome, banco vinculado, limite, dia de fechamento e dia de vencimento da fatura.

  • Ao lançar uma despesa, selecione o cartão no campo correspondente
  • Compras parceladas: informe o número de parcelas — o sistema agenda todas
  • Em Faturas no dashboard você vê o total de cada fatura por mês
  • O saldo disponível do cartão é atualizado automaticamente

🔄 Lançamentos Recorrentes

Para despesas e receitas que se repetem mensalmente (aluguel, salário, planos, assinaturas), use a aba Recorrentes:

  • Cadastre uma vez e o sistema lembra todo mês
  • Um alerta aparece no dashboard quando houver recorrentes pendentes no mês atual
  • Ao clicar em Lançar, um lançamento é criado automaticamente com os dados cadastrados
  • É possível pausar ou encerrar um recorrente a qualquer momento

🏦 Financiamentos e Empréstimos

Na aba Financiamentos, cadastre qualquer tipo de dívida parcelada (financiamento de carro, empréstimo pessoal, consórcio etc.):

  • Valor Total Financiado: Capital tomado emprestado (PV)
  • Taxa de Juros: % ao mês (deixe 0 para parcelamento sem juros)
  • Nº de Parcelas: Total de prestações
  • Parcelas Já Pagas: Se o financiamento já existe, informe quantas já foram quitadas
  • Data da 1ª Parcela: Data de vencimento da primeira prestação

O sistema usa a Tabela Price para calcular a parcela: PMT = PV × [i×(1+i)^n] / [(1+i)^n − 1]. O preview aparece em tempo real antes de salvar.

Ao salvar, as parcelas restantes são lançadas automaticamente em Lançamentos com as datas corretas. Acesse o botão Parcelas no card do financiamento para:

  • Ver todas as parcelas com status (Paga / Próxima / Futura)
  • Lançar a próxima individualmente ou todas de uma vez
  • Ajustar o valor das parcelas futuras (ex: reajuste por índice)
  • Simular a quitação antecipada com desconto de juros futuros

📈 Investimentos e Patrimônio

Na aba Investimentos, cadastre seus ativos financeiros (CDB, Tesouro Direto, ações, fundos, poupança etc.):

  • Informe nome, tipo, banco, valor aportado, valor atual e rentabilidade
  • Ao cadastrar com banco e data, um lançamento de aporte é criado automaticamente
  • Use o botão + Movimentação para registrar aportes, resgates e rendimentos
  • Cada movimentação gera um lançamento vinculado em Lançamentos
  • O dashboard exibe o total aportado, patrimônio atual e rentabilidade geral

🎯 Metas Financeiras

Defina objetivos financeiros na aba Metas. Para cada meta informe:

  • Descrição (ex: "Reserva de emergência", "Viagem para Europa")
  • Valor alvo e valor já acumulado
  • Data limite para atingir o objetivo

O sistema calcula o percentual de progresso e o valor mensal necessário para atingir a meta no prazo. Você também pode definir limites de gasto por Centro de Custo — um alerta aparece no dashboard quando o limite for ultrapassado.

📊 Dashboard — Visão Geral das Finanças

O Dashboard é a tela inicial com todos os indicadores importantes. Use o filtro no topo para selecionar o período (mês, ano ou próximos 6 meses) e a conta bancária:

  • Saldo Total Hoje: Soma dos saldos de todas as contas considerando apenas lançamentos pagos até hoje. Representa seu dinheiro real disponível agora.
  • Saldo do Período: Receitas menos Despesas do período selecionado. Inclui lançamentos futuros agendados até o final do filtro — mostra como suas finanças vão ficar.
  • Total Receitas / Despesas: Totais do período selecionado
  • Gráfico de Fluxo: Evolução diária de entradas e saídas
  • Faturas dos Cartões: Alertas de faturas próximas do vencimento
  • Comparativo mensal: Comparação com o mês anterior

📋 Relatórios

Na aba Relatórios, gere e exporte análises em PDF:

  • Fluxo de Caixa Mensal: Movimentações dia a dia com totais
  • Movimentação Bancária por Conta: Extrato de cada banco
  • Movimentação por Cartão: Gastos por cartão de crédito
  • Despesas por Categoria: Ranking de gastos por centro de custo
  • Receitas vs Despesas: Comparativo em período personalizado
  • Contas a Vencer / Em Atraso: Inadimplências e próximos vencimentos
  • Posição de Investimentos: Carteira consolidada

⚖️ Conciliação Bancária

A conciliação compara o extrato do banco com seus lançamentos e identifica divergências. Acesse a aba Conciliação Bancária:

OFX / QFX / CSV — Baixe o extrato no internet banking do seu banco (procure por "Exportar extrato") e importe aqui. O sistema compara automaticamente e apresenta as diferenças uma a uma para você confirmar ou pular.

PDF / Texto — Selecione o banco (Itaú, Bradesco, Caixa ou Nubank), faça upload do PDF ou cole o texto copiado. O sistema extrai as transações com o parser específico de cada banco. PDFs escaneados da Caixa usam OCR via IA automaticamente.

Após processar, uma tabela mostra todas as transações com status OK (já lançadas) ou Novo (não encontradas). Clique em Iniciar Conciliação para confirmar os novos lançamentos — o sistema usa tolerância de ±3 dias na comparação de datas.

💾 Backup e Segurança dos Dados

  • Em Configurações → Backup Manual, exporte um arquivo JSON com todos os dados
  • Ative o Backup Automático (diário, semanal ou mensal) para nunca perder dados
  • Para restaurar, clique em Importar Backup e selecione o arquivo JSON
  • Faça backup antes de limpar dados ou trocar de dispositivo
  • Se precisar apagar tudo, use Configurações → Limpar Todos os Dados — confirme no modal e a página recarregará limpa

🔒 Proteção e Criptografia

Todos os dados são cifrados com AES‑256 usando sua senha pessoal como chave. Sem a senha correta, os dados aparecem como texto embaralhado ilegível. A criptografia acontece localmente no dispositivo — nenhum dado é enviado para servidores externos.

  • Nunca compartilhe sua senha
  • Se trocar de dispositivo, exporte o backup e importe no novo — use a mesma senha
  • O sistema não possui mecanismo de recuperação de senha — guarde-a em local seguro

💡 Dicas de Uso

  • Use Ctrl+K (desktop) para abrir a busca global rápida
  • Lance despesas pendentes com antecedência para planejar o mês
  • Revise o dashboard toda semana para acompanhar as metas
  • Faça a conciliação bancária mensalmente para garantir que nada foi esquecido
  • Use centros de custo consistentes — isso torna os relatórios muito mais úteis
  • O botão Duplicar no lançamento facilita registrar despesas similares rapidamente
  • Lançamentos recorrentes poupam tempo com contas mensais fixas
  • Sempre exporte um backup antes de atualizar o aplicativo
