Guia de uso completo

Visão Geral

O Nobrega Mall Consultant – Finance é um sistema completo de controle financeiro que funciona 100% offline no navegador. Todos os dados são armazenados no localStorage, sem necessidade de internet após o carregamento inicial.

Login

  • Use admin / 1234 para o acesso padrão.
  • Novos usuários podem ser criados na aba Usuários.

Dashboard

  • Filtre por mês e banco para ver os indicadores do período.
  • O gráfico de Fluxo de Caixa mostra a variação diária de receitas e despesas.
  • Os gráficos de pizza mostram a distribuição por centros de custo.

Lançamentos

  • Registre receitas e despesas com tipo, data, conta corrente, centro de custo e forma de pagamento.
  • Para pagamentos com cartão, selecione o cartão e informe o número de parcelas.
  • Use os filtros na tabela para buscar por qualquer campo.
  • Clique nos cabeçalhos para ordenar por coluna.

Cartões de Crédito

  • Cadastre cartões com limite total, dia de fechamento e vencimento.
  • O limite utilizado é calculado automaticamente com base nos lançamentos feitos no cartão.
  • O limite disponível = Limite Total − Utilizado.

Cadastros

  • Bancos: cadastre suas contas correntes/bancos.
  • Centros de Custo: categorias como Alimentação, Salário, etc.
  • Formas de Pagamento: PIX, Boleto, Transferência, Cartão, Dinheiro, etc.

Configuração

  • Faça backup exportando os dados como JSON.
  • Importe um backup para restaurar os dados.
  • Use com cuidado a opção de limpar dados.
